To install Pac-Man for Microsoft Windows, download an OpenJDK binary and then download the desired version. Double click the .exe file and follow the installer's instructions.

Note: While Microsoft and Apple are suspicious of Pac-Man, it is not a virus. You may scan it with an antivirus program to confirm this.

To install on Mac, download Java 8 and then the desired version. Double-click the downloaded .zip file, open the created folder, and double click the Pac-Man file.

If the Java 8 link does not work, click here.

Are you unable to download it? No problem! Click here to play from your browser. Note: The web version does not recieve bug fixes, and has fewer features than the installed version.
